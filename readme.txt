== Description ==

WP Nice Scroll is a fully customizable wordpress scrollbar plugin which enable you to change scrollbar color, border color, radius whatever you want.

<p><strong>Plugin Features </strong></p>

<ul>
<li>Very lightweight</li>
<li>Super easy installation.</li>
<li>Unlimited colors.</li>
<li>All browser supported</li>
<li>Touch supported</li>
<li>Scrollbar border radius.</li>
<li>Scrollbar Width.</li>
<li>Scrollbar scroll speed.</li>
<li>Scrollbar border.</li>
<li>Auto Hide Enable/Disable Mode.</li>
<li>Easy Option Panel.</li>
<li>Easy Documentation.</li>
</ul>

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

01. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
02. Activate the plugin through the 'Plugins' screen in WordPress.
03. Use the Settings->WP Nice Scroll Settings screen to configure the plugin.
